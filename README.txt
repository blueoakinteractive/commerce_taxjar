## Purpose

This module provides integration between Drupal Commerce
and [TaxJar](http://affiliate.taxjar.com/blueoakinteractive).
Note, an affiliate link is used in all references to TaxJar to help
fund the development of this module. Please consider registering
with TaxJar via the affiliate URL.

## Installation

* Download the module to your sites module directory
* Download the composer_manager module
* Enable the composer_manager module
* Enable the commerce_taxjar module
* Visit sites/default/files/composer and run "composer update"
